## 想定質問と回答

| 質問 | 回答 |
|------|------|
| **データはどこに保管？** | Google Cloud東京リージョン（asia-northeast1）のみ。海外転送なし |
| **AIにデータが学習されない？** | Google Vertex AIの契約条件で学習不使用。プライバシーポリシーにも明記 |
| **バックアップは？** | Firestore日次自動バックアップ、7日間保持。リストア手順書あり |
| **障害時は？** | Cloud Monitoringで3種のアラート（エラー率/レイテンシ/メモリ）を自動検知。運用マニュアル完備 |
| **費用は？** | Cloud Run従量課金（未使用時0円）。Vertex AI は API呼び出し回数に応じた課金 |
| **導入にどのくらいかかる？** | 許可メール設定のみで即日利用開始可能。職員登録は初回ログインで自動 |
| **他の自治体にも展開できる？** | 現在は単一自治体向け設計。マルチテナント対応は将来拡張可能 |
| **オフラインでは使える？** | クラウドサービスのためオンライン接続が必要 |
| **監査は？** | 全APIリクエストの監査ログを自動記録（個人情報は非出力）。Cloud Loggingで検索可能 |
| **PDFの帳票は？** | 現在はブラウザの印刷機能で代替。PDF帳票生成は今後の機能拡張として計画中 |
